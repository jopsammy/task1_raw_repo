## 阶段 S0: (q,t)-Catalan 递推 — 数学结构验证

### 🟢 Task S0-1：Garsia-Haglund 递推验证 ✅ 已完成（2025-05-24）

**成果**：验证 Garsia-Haglund (q,t)-Catalan 递推公式。

$$H_{n,n}=a^{\binom{n}{2}},\quad H_{n,k}=b^{n-k}a^{\binom{k}{2}}\sum_{r=1}^{n-k} \binom{r+k-1}{r}_a H_{n-k,\,r}$$

$$F[n]=a^{n(n+1)/2}\cdot b^{-n}\cdot H_{n+1,1} \pmod{998244353}$$

n≤8 四组 (a,b) 全量验证通过（`verify_bergeron.py`）。

**复杂度评估**：O(N³/6)。N=500 实测 3.89s。N=5000 推算 ~65min。不满足 O(N²) 目标。

**止损判定**：公式正确但复杂度不达标。与其他两条路径（L(i) DP O(N³/2)、Route D O(N³)）收敛于同一瓶颈——q-binomial 加权求和不可降阶。

**同时确认**：
- b=1 Garsia 递推：O(N²)，正确但仅适用纯 alice
- b=a⁻¹ 闭合式：O(N)，仅特例
- bounce path 分解：segment≠x_seq，area 不可由 segment 长度确定，未突破瓶颈
- 对称性：C_n(a,b)=C_n(b,a)

### 🟢 Task S0-6：v14 NTT 卷积降阶 🆕 ✅ 已完成（2025-05-24）→ 🔧 第二轮判题后修复（2025-05-24）

**成果**：通过 q-阶乘变量分离将 Garsia-Haglund 的 O(N³/6) 瓶颈化为 O(N² log N)。

**第二轮判题反馈**：7/21 正确，14/21 答案错误。
- **根因**：q-阶乘在 a 有小阶（如 a=MOD-1, ord=2）时产生零因子 → `0/0 = 0` 而非正确值
- **修复**：双算法架构 —— 预检测 `(a^i-1)≡0`，若有零因子则回退到 q-二项式递推（无除法）+ 8路ILP
- **验证**：128 次 Python 镜像测试全量通过（50 NTT + 40 回退 + 48 BF)

**核心数学**：

$$H_{m+k,k} = b^m a^{\binom{k}{2}} \sum_{r=1}^m \binom{r+k-1}{r}_a H_{m,r}$$

$$\downarrow \binom{r+k-1}{r}_a = \frac{\text{Fact}[r+k-1]}{\text{Fact}[r] \cdot \text{Fact}[k-1]}$$

→ 翻转 A 序列与 B 序列做 NTT 卷积，每 m 一步 O(N log N)，总 O(N² log N)。

### 🟢 Task S0-7：v15 判题反馈 (19/21) → v15-b 修复 → **21/21 判题通过** ✅

**目标**：通过 q-Lucas 分解 + 二维 NTT 将 v14 的 O(N³/6) 回退路径升级为 O(N² log N)，统一全 a,b 为单一算法。

**核心数学**（已通过 GN-004 独立审查，修正了 q-Lucas 索引偏移与借位情形）：

设 a 的乘法阶为 d。q-Lucas 分解（r=r₁d+r₀, k=k₁d+k₀）：
$$n = r+k-1 = (r_1+k_1)d + (r_0+k_0-1)$$
- k₀=0, r₀=0（借位）：binom(r₁+k₁-1, r₁) · binom(d-1, 0)_a = binom(r₁+k₁-1, r₁)
- k₀=0, r₀>0：0（n₀ = r₀-1 < r₀）
- k₀≥1, r₀+k₀≤d：binom(r₁+k₁, r₁) · binom(r₀+k₀-1, r₀)_a
- k₀≥1, r₀+k₀>d（进位）：0
- d=1 时 a≡1，走对称性 b=1 Garsia O(N²) 特化

→ 余数层互相关（NTT 大小 2d）+ 商层互相关（NTT 大小 2N/d）→ O(N² log N)。

## 开发背景

### 项目背景

需要一个全局控制器来协调Pipeline的执行，提供统一的接口来启动、暂停、继续、停止Pipeline，并支持回滚到指定阶段。

### 技术选型

- **线程控制**: 使用threading.Event实现暂停/继续
- **单例模式**: 确保全局只有一个PipelineController实例
- **状态机**: 使用枚举管理控制器状态

---

## 架构设计

### 类结构

```
PipelineController (主类)
  ├── start_pipeline() - 启动Pipeline
  ├── pause_pipeline() - 暂停Pipeline
  ├── resume_pipeline() - 继续Pipeline
  ├── stop_pipeline() - 停止Pipeline
  ├── rollback_to_stage() - 回滚到指定阶段
  ├── get_status() - 获取状态
  ├── get_logs() - 获取日志
  ├── get_results() - 获取结果
  ├── get_result() - 获取指定阶段结果
  ├── clear() - 清空状态
  ├── _log() - 记录日志
  ├── _update_progress() - 更新进度
  ├── _should_stop() - 检查是否应该停止
  ├── _wait_for_resume() - 等待恢复
  ├── _handle_stop() - 处理停止
  └── _handle_failure() - 处理失败

ControllerStatus (枚举)
  ├── IDLE
  ├── RUNNING
  ├── PAUSED
  └── STOPPED
```

### Pipeline阶段顺序

1. **需求锚定** (requirement_anchoring) - 10-20%
2. **需求校验** (requirement_validation) - 30-40%
3. **架构迭代** (architecture_iteration) - 50-70%
4. **契约生成** (contract_generation) - 80-90%
5. **IDE引导包生成** (ide_bundle_generation) - 95-100%

## 核心实现

### 1. PipelineController 初始化

```python
def __init__(self, pipeline_orchestrator=None):
    self.pipeline_orchestrator = pipeline_orchestrator or get_pipeline_orchestrator()
    self.status = ControllerStatus.IDLE
    self.current_stage = ""
    self.progress = 0
    self.logs = []
    self.results = {}
    self.start_time = None
    self.end_time = None
    
    self._pause_event = threading.Event()
    self._pause_event.set()  # 默认不暂停
    self._stop_event = threading.Event()
```

### 2. start_pipeline 执行流程

```
1. 检查是否正在运行
2. 初始化状态（status=RUNNING, logs清空, results清空）
3. 记录启动日志
4. 按顺序执行5个阶段：
   a. 检查是否应该停止 (_should_stop)
   b. 等待恢复 (_wait_for_resume)
   c. 更新进度
   d. 调用pipeline_orchestrator执行阶段
   e. 保存结果
   f. 检查是否成功
   g. 更新进度
5. 标记完成，记录完成日志
6. 返回结果
```

### 3. 暂停/继续机制

```python
# 暂停
def pause_pipeline():
    self.status = PAUSED
    self._pause_event.clear()

# 继续
def resume_pipeline():
    self.status = RUNNING
    self._pause_event.set()

# 等待恢复
def _wait_for_resume():
    self._pause_event.wait()
```

### 4. 停止机制

```python
# 停止
def stop_pipeline():
    self._stop_event.set()
    self._pause_event.set()  # 唤醒等待

# 检查是否应该停止
def _should_stop():
    return self._stop_event.is_set()
```
